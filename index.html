FRONTEND DEVELOPER

Diventa un front-end developer e scopri un mondo di opportunità

Gli sviluppatori front-end lavorano a stretto contatto con i designer per creare siti web e applicazioni belle, funzionali e veloci. Percorso

1 / HTML & CSS

Impara a sviluppare siti completamente da zero. Con HTML e CSS sarai in grado di implementare la struttura e lo stile di una pagina web. Pratica con le tecnologie più moderne, come Flexbox, Grid e tanto altro.

Basi di HTML & CSS

2 / CSS avanzato

Porta le tue skill al prossimo livello. Crea splendide e interattive animazioni, sfrutta SASS per implementare lo stile in modo semplice e rapido.

CSS Avanzato

Crea splendide interfacce con SASS, CSS Grid e tanto altro

3 / Javascript

Impara uno dei linguaggi di programmazione più utilizzati al mondo. Manipola il DOM, gestisci Javascript asincrono, utilizza la programmazione orientata agli oggetti. Conoscerai le versioni più moderne di JS per diventare uno sviluppatore completo

Javascript

Impara uno dei linguaggi più versatili, potenti e ricercati al mondo

4 / React

Utilizza React, la libreria creata da Facebook, per sviluppare delle web app complete e performanti. Insieme ad altre tecnologie, come React Router, Redux e Firebase, potrai creare tutto ciò che vuoi

React

Crea complesse web app con la tecnologia più in voga del momento

5 / React Native

Impara React Native, il framework basato su React, che ti permette di creare applicazioni per iOS e Android, condividendo quasi tutto il codice. Dai vita a splendide app mobile e pubblicale negli store per raggiungere il tuo obiettivo

React Native

Sviluppa applicazioni iOS e Android condividendo quasi tutto il codice

FAQ / Domande Frequenti

Devo saper programmare per cominciare il percorso?

No, il curriculum è stato creato per dare la possibilità a tutti di poter iniziare. Molti utenti hanno cominciato completamente da 0

Qual è la differenza tra il piano BASE e il piano PRO?

Il piano base offre una parte dei video corsi, con l’aggiunta dei quiz. Col piano PRO hai i video corsi completi e varie feature avanzate, come revisione del tuo codice, accesso alla community, certificati e possibilità di trovare lavoro

Perchè un abbonamento?

I corsi non comprendono solamente i video, ma diamo un supporto costante tramite la nostra community e la revisione del codice. Tramite un abbonamento mensile, siamo in grado di tenere il costo totale accessibile a tutti

Cos’è un percorso e come funziona?

I percorsi sono dei curriculum che comprendono al loro interno diversi corsi. Sono stati creati per insegnare tutte le competenze necessarie per inserirsi in un campo, come lo sviluppo frontend
